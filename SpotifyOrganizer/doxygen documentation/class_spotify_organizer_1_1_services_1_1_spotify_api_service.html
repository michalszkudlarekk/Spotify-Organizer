Detailed Description

The SpotifyApiService class contains a constructor that validates credentials and creates a SpotifyClient object. SearchTrack is a method that takes a string as input and returns a FullTrack object. It creates a SearchRequest object using the input and sets the limit to 1, which means that the method will return the first track that matches the search query.
