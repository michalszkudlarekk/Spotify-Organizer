class  SpotifyOrganizer.Models.AlbumSong
 The AlbumSong class is a model that represents the many-to-many relationship between the Album and Song models in the database. This class includes a foreign key reference to the SongId and AlbumId properties, which allows us to retrieve information about which song belongs to which album. The properties are decorated with the Required attribute to ensure that they are not null or empty.
